Introdution

本次作業目的在於判斷垃圾郵件，至少需做出兩種方法，其中一種必須為logistic regression。

作業使用的data已經幫你從信件抽取相關的特徵，利用這些特徵去判斷是否為垃圾郵件。

spam_train.csv : 訓練資料 4001筆

spam_test.csv:   測試資料 600筆

特徵的第一維是ID,最後一維是label。其他維度請看spambase.names。
